# Folder Upload Foto Siswa

Silakan upload foto siswa ke folder ini.

## Contoh penamaan file
- `0071717838.jpg`
- `0071752502.jpg`

## Catatan
- Gunakan nama file sesuai NISN agar mudah dipetakan.
- Setelah upload, pastikan field `foto` di `data.json` mengarah ke file yang benar, misalnya:
  - `"foto": "photos/0071717838.jpg"`
